# dashboard_de_jogos

Um dashboard de vendas globais de jogos utilizando como interface o streamlit, pandas para a analise de dados e plotly para a criação dos gráficos.

# Como Executar o Programa

Siga os passos abaixo para configurar o ambiente e rodar o dashboard em sua máquina.

1. Instalação das Dependências

Antes de iniciar, você precisa instalar as bibliotecas Python necessárias. Abra o seu terminal e execute o comando:

pip install pandas streamlit plotly

2. Inicialização do Dashboard

Com as bibliotecas instaladas, navegue até a pasta onde o arquivo está salvo e execute o seguinte comando:

streamlit run dashboard_jogos.py

3. Visualização

Após rodar o comando, o Streamlit iniciará um servidor local. O dashboard deverá abrir automaticamente em uma nova aba do seu navegador padrão. Caso não abra, utilize o link (URL) que aparecerá no seu terminal (geralmente http://localhost:8501).
